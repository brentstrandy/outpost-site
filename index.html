Grid

The grid expands to fit whatever context is relevant. This width is 960px for the homepage, courses, pricing etc. For the dashboard, & course player, this width is 100% (minus 60px margin on each side) of the viewport. Here, it is limited to 1200px for demonstration only.

These layout guides are only generally followed within the initial design phase. Whenever there is a conflict, fall back to the PSD. For pages created after the original design phase, follow this grid in order to ensure the best UX.
